# figma-welcome-impl

情侣吵架复盘 App 欢迎页静态实现（HTML + CSS）。

## 本地预览

直接双击 `index.html`，或使用任意静态服务器。

## 部署到 Render

点击下面按钮可一键从本仓库创建 Render 服务：

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/drew131382-create/figma-welcome-impl)

Blueprint 配置见 `render.yaml`。
